# Components

Компонент `Components` не имеет визуальных особенностей, используется для быстрого вывода набора компонентов.

```php
make(iterable $components = [])
```

- `$components` - массив компонентов.

```php
use MoonShine\UI\Components\Components;

Components::make([
    // components
]);
```
